*args and **kwargs

2016-11-23

It is not necessary to write in the format of (args and **kwargs), only the (aesteric) is essential.

Both args and **kwargs allow you to pass a variable number of arguments to a function, but you don’t know how many arguments will be passed to your function. args is used to send a non-keyworded variable length argument list, while **kwargs is used to pass keyworded arguments(dict).
